## 這是什麼

這個 repo 本身就是一個示範：你看到的網頁，底層是知識庫（markdown），再加一份 SKILL.md 的 Agent 指令，接上 AI 就會跑。這就是「文件即系統」。

這是江江教練的個人 AI 名片。它有兩種讀法：

- 人看：讀 `profile.md`，或之後的網頁版。
- AI 讀：把這個 repo 或連結丟進你的 ChatGPT、Claude，它會讀 `SKILL.md` 或 `llms.txt` 的指令，先徵得你同意，再以江昱德的商務 AI 分身身份，依你的身份替江江做客製的自我介紹。

## 內容

- `profile.md`：公版自我介紹
- `SKILL.md`：給 AI 的使用說明，這張名片的腦
- `llms.txt`：給 AI 的完整接待協議與資訊，純丟網址、零提示也讀得到
- `intros/`：各行各業的客製介紹版本
