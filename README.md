## 這是什麼

這是江江教練的個人 AI 名片。它有兩種讀法：

- 人看：讀 `profile.md`，或之後的網頁版。
- AI 讀：把這個 repo 或連結丟進你的 ChatGPT、Claude，它會讀 `SKILL.md` 的指令，依你的身份替江江做客製的自我介紹。

## 內容

- `profile.md`：公版自我介紹
- `SKILL.md`：給 AI 的使用說明，這張名片的腦
- `intros/`：各行各業的客製介紹版本
